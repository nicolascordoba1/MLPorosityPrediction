# Estimación de la porosidad de yacimiento de hidrocarburos con aprendizaje automático usando sísmica post-stack 

Este repositorio contiene los productos de la tesis de maestría en geofísica de Nicolas Cordoba acerca de la estimación de la porosidad de yacimiento de hidrocarburos con aprendizaje automático usando sísmica post-stack. El contenido de este repositorio se encuentra en constante cambio hasta la finalización de la tesis.

## Tabla de Contenido

- [Uso](#uso)
- [Contact Information](#información-de-contacto)

## Instalación

Para instalar las librerías usa `pip install -r requirements.txt`

## Uso

Todos los códigos se encuentran escritos en python en el formato de Jupyter Notebook para que sea mas sencilla su reproducibilidad. Dentro de cada uno de ellos se encuentra una breve documentación de las funciones. En la carpeta de modelos se encuentran las funciones usadas para las simulaciones númericas para estimar porosidad a partir de sísmica. En la carpeta ML se encuentran los modelos probados para la investigación y la propuesta de arquitectura de red neuronal propia para estimar la porosidad. Los datos usados corresponden con la base de datos libre de Terranubis.
